# 🏢 Warehouse Management System

A full-stack Warehouse Management System built using **ASP.NET Core MVC**.  
The system is designed to manage warehouse items with a secure, role-based approval workflow.

---

## ✨ Features

- 🔐 JWT-based authentication and secure login system  
- 👥 Role-based authorization:
  - Admin  
  - Warehouse Manager  
  - General Manager  
- 📦 Full CRUD operations for warehouse items  
- 🔄 Multi-level approval workflow:
  - Item creation  
  - Warehouse Manager review  
  - General Manager final approval or rejection  
- 🗄️ SQL Server database integration  
- ⚡ Efficient data handling using Entity Framework Core  
- 🏗️ Scalable and clean application structure  

---

## 🛠️ Technologies Used

- ASP.NET Core MVC  
- C#  
- Entity Framework Core  
- SQL Server  
- JWT Authentication
